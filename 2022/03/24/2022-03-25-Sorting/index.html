Sorting

Posted on 2022-03-25 In Algorithm Views:
  • 정렬 비교

Bubble Sort

  • 시간 복잡도 : n(n-1)/2 => O(N^2)
  • 공간 복잡도 : O(N)
  • Stable sort.

Selection Sort

  • 시간 복잡도 : n(n-1)/2 => O(N^2)
  • 공간 복잡도 : O(N)
  • Unstable sort.

Insertion Sort

  • 시간 복잡도 : best case = O(N), worst case = O(N^2)
  • 공간 복잡도 : O(N)
  • Stable sort.

Merge Sort

  • LinkedList의 정렬에 효과적.
  • 시간 복잡도 : O(NlogN)
  • Stable sort

Quick Sort

  • 시간복잡도 : O(Nlog2N)
  • 공간복잡도 : O(N)
  • Unstable sort
